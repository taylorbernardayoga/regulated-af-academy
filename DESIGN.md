# Design system

Plain-language rules for every visual decision. The goal is a warm, earthy, editorial look that feels grounded and bold, not clinical or corporate.

## Color tokens

- `--rust` #B5491F: primary accent. Buttons, links, key emphasis.
- `--terracotta` #C9663A: secondary warm accent, hovers, highlights.
- `--gold` #C99A3F: small accents, eyebrows, fine details.
- `--bone` #F4EFE6: page background, light surfaces.
- `--cream` #FBF8F2: cards and raised surfaces on the bone background.
- `--ink` #2B2420: body text, headlines.
- `--clay` #6E5F54: muted secondary text.
- `--forest` #3B4A3F: deep feature section backgrounds.

When to use each: rust is the one loud color, spend it on the main actions. Bone is the default canvas. Forest is for the one dark feature section per page. Keep everything else quiet.

## Type

- Display: a high-contrast serif (Fraunces) for headlines, used with restraint.
- Body: a clean sans-serif (Inter) for everything readable.
- Eyebrows: small uppercase Inter with wide letter spacing, in gold or clay.

Type scale: big confident headlines, generous body size (around 18px), clear hierarchy. One signature headline per page, the rest calmer.

## Components

- Buttons: rust background, cream text, generous padding, slight rounding. Hover shifts to terracotta.
- Cards: cream background on bone, soft shadow, comfortable padding.
- Eyebrows: small uppercase label above a headline.
- Leads: slightly larger intro paragraph under a headline, in clay.
- Reveal: gentle fade-and-rise as sections scroll into view.

## Spacing and rhythm

Generous vertical space between sections. Let things breathe. Content sits in a centered column, not edge to edge.

## Motion

Subtle only. Scroll reveals and button hovers. Respect reduced-motion settings. No bouncing, no spinning, nothing that screams template.

## Patterns

- Dark feature section: one forest-background section per page for the most important emotional beat.
- One signature per page: pick a single memorable element and keep everything else quiet around it.

## Guardrails

Match this system on every new page. Never use em-dashes in any copy.
